# Popcorn Film Log — Authentication App with Dark Cinema Theme

## Features

- **Login screen** where users can sign in with either their email or username, plus password
- **Create Account screen** where new users register with a username, email, and password (with confirmation)
- **Password visibility toggle** on password fields for convenience
- **Input validation** — checks for valid email format, password length, matching passwords, and non-empty username before submitting
- **Error messages** displayed clearly when login or registration fails (wrong password, account already exists, etc.)
- **Loading indicators** during authentication requests
- **Automatic session persistence** — stays logged in between app launches using saved token
- **Home screen** shown after login with a welcome message and sign-out option
- **Smooth animated transitions** between login, registration, and home screens

## Design

- **Dark cinema theme** — deep black/charcoal backgrounds with warm amber/gold accent colors, evoking a movie theater atmosphere
- **Popcorn branding** — app name "Popcorn" with a popcorn emoji/icon and a film-related tagline
- **Large, bold title** at the top of each screen with a warm gradient text effect
- **Rounded input fields** with subtle dark borders on a slightly elevated background
- **Full-width amber/gold primary buttons** with a pressed animation
- **Secondary text links** for switching between Login and Create Account
- **Subtle haptic feedback** on button taps
- **Staggered fade-in animations** when screens appear
- **Semantic dark mode colors** that feel like a premium cinema experience

## Screens

1. **Login Screen** — App logo/title at top, email-or-username field, password field with visibility toggle, "Sign In" button, "Don't have an account? Create one" link at bottom
2. **Create Account Screen** — Title, username field, email field, password & confirm password fields with visibility toggles, "Create Account" button, "Already have an account? Sign In" link
3. **Home Screen** — Welcome message showing the user's username, profile placeholder, and a "Sign Out" button — serves as a landing page after authentication

## App Icon

- A dark background with a stylized popcorn bucket in warm gold/amber tones, with a subtle film reel or cinema motif — clean, modern, and recognizable
